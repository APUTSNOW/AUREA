# AUREA Technical Blueprint

## Moduli principali

### UI Layer
Gestisce le schermate e i componenti visivi.

### Treasure Engine
Carica i dati da `data/songs.json` e costruisce il Tesoro associato alla canzone.

### Echo Engine
Modulo futuro: genererà riflessioni e prospettive senza presentarle come verità assolute.

### Journey Engine
Modulo futuro: salverà le scoperte del Viaggiatore.

### Golden Thread
Modulo futuro: collegherà canzoni diverse attraverso temi, simboli ed emozioni comuni.

## Principio tecnico

Ogni nuova canzone dovrà poter essere aggiunta modificando `songs.json`, senza riscrivere l'interfaccia.
